# AirBnb-Dashboard
Project Overview

The Airbnb Performance Dashboard is an interactive Power BI dashboard designed to analyze Airbnb listings across multiple cities and uncover key trends in pricing, ratings, property availability, and overall market performance.

The dashboard enables users to compare Airbnb markets, identify high-performing locations, and explore customer satisfaction trends through dynamic visualizations and KPI tracking.

🎯 Purpose

The dashboard provides a comprehensive view of Airbnb market performance by analyzing over 279K+ listings across 10 cities.

It helps answer critical questions such as:

Which cities have the highest-rated Airbnb properties?
How do pricing trends vary across locations?
Which markets offer the best balance between price and ratings?
What patterns exist in customer reviews and property performance?

This dashboard is useful for:

Airbnb Hosts
Property Investors
Travel & Hospitality Analysts
Market Researchers
Data Analytics Enthusiasts
🛠️ Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Primary data visualization platform
📂 Power Query – Data cleaning and transformation
🧠 DAX (Data Analysis Expressions) – KPI calculations and analytical measures
🔗 Data Modeling – Relationship management and optimized reporting
📈 Interactive Visualizations – Charts, KPI Cards, Maps, and Slicers
📁 PBIX File Format – Dashboard development and reporting

⭐ Features & Highlights
Business Problem

The Airbnb marketplace contains a large volume of listing data spread across different cities and regions.

Property owners, investors, and analysts often face challenges when trying to identify:

High-performing markets
Competitive pricing strategies
Customer satisfaction trends
Opportunities for investment

Analyzing raw listing data manually is inefficient and time-consuming.

🎯 Goal of the Dashboard

The objective of this dashboard is to:

Analyze Airbnb market performance across cities
Compare pricing and customer ratings
Identify top-performing locations
Support investment and business decisions
Deliver actionable insights through interactive visualizations

💡 Key Insights
Certain cities consistently achieve higher ratings despite premium pricing.
Listing concentration varies significantly across markets.
Customer satisfaction and pricing do not always move together.
Market performance differs based on location-specific demand patterns.
